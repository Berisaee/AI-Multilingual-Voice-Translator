# 🌍 AI Multilingual Voice Translator
An AI-powered multilingual voice translation application built using **Python**, **Streamlit**, **Speech Recognition**, **Deep Translator API**, and **Text-to-Speech (gTTS)**.

The application allows users to:
* Speak in one language
* Convert speech to text
* Translate text into multiple languages
* Listen to translated speech output
* Download translated audio files

---

# 🚀 Features
- ✅ Real-time Speech Recognition
- ✅ Automatic Language Translation
- ✅ Text-to-Speech Audio Generation
- ✅ Editable Recognized Text
- ✅ Audio Playback Support
- ✅ Download Translated Audio
- ✅ Multi-language Support
- ✅ Clean Streamlit UI
- ✅ Modular Python Architecture

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Berisaee/AI-Multilingual-Voice-Translator.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd AI-Multilingual-Voice-Translator
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Application

```bash
streamlit run app.py
```
